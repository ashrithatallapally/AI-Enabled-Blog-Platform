# AI-Enabled Blog Platform

A simple full-stack web application that generates blog content using AI.
This project uses a React frontend and a Node.js backend to demonstrate how AI APIs can be integrated into real-world web applications.

## Features
- AI-based blog content generation
- React frontend
- Node.js + Express backend
- REST API communication

## Technologies Used
- React
- Node.js
- Express
- Axios
- Gemini API

## Purpose of the Project
This project was built as a learning experience to understand full-stack development and AI integration.

## Current Status
Core functionality implemented. More features and UI improvements planned.
